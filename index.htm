Этот плагин представляет собой простое решение для сайта на котором нужно реализовать большое количество всплывающих окон. Он сделан максимально просто, без предустановленых стилей и размеров что позволяет легко кастомизирывать его под любые нужды. Позволяет вызывать всплывающие окна находясь уже в открытом всплыающем окне. Для подключения не требуется много усилий, достаточно подключить 2 файла. Для инициализации плагина достаточно добавить кнопку и всплывающее окно с датаатрибутами которые указаны в script.js
